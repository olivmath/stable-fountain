## Como seus clientes movimentam dinheiro hoje? Descreva o fluxo completo.
Investidor:
- Cadastro
- Aprovação KYC (aqui tem burocracia de limites de investimento)
- Pix
- Recebe stable BR na wallet
- Executa a compra do token

Emissor:

- Gerencia o saldo durante a captação
- Faz offramp para pagar originador
- Faz onchain para pagar investidor (dividendos de acordo com frequencia de investimento)

## Descreva como você imagina que a solução funcionaria (depósito → resgate).
exatamente como uma stablefunciona, porém o onramp que seria controlado por nós.

onramp:
o investidor faz pix para uma conta bancária nossa
identificamos o pix e compramos stable BR
com stable em custódia, executamos o mint
o investidor recebe o token em wallet

offramp:
o investidor solicita o offramp
o token é transferido para nossa wallet emissora/hot
queimamos o token e vendemos stable BR
entregamos o saldo via pix na wallet do investidor
